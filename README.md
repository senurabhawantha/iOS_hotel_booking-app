## 🧠 What This App Does

The **Hotel Booking iOS App** is a mobile application designed to let users:

- Search and explore hotel listings
- View detailed information (amenities, price, location)
- Select check-in and check-out dates
- Choose rooms and confirm bookings
- See reservations and booking summaries

This project is ideal for showcasing your skills in Swift, UI design, user interaction, and data handling.

---

## 📌 Core Features (Explained)

### 🔎 **Hotel Search**
Users can search for hotels by location or hotel name. The search results update as they type, providing a seamless discovery experience.

---

### 🏨 **Hotel Details Page**
Each hotel includes:

- Photos / gallery
- Price per night
- Ratings / reviews
- Amenities list
- Location details

This screen helps users make informed choices.

---

### 📅 **Date & Room Selection**
Users can pick:

- Check-in / Check-out dates
- Number of guests
- Room type (e.g., single, double, suite)

The UI guides users through making a booking step by step.

---

### 📜 **Booking Confirmation**
After choosing dates and rooms:

- Users can review booking details
- Confirm reservations
- View a booking summary screen

This flow mimics real-world hotel booking UX.
